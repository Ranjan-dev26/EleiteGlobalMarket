# EleiteGlobalMarket React Website

A premium broker-style React landing page inspired by the provided reference websites.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Important

Update the login and signup URLs in `src/App.jsx`:

```js
const loginUrl = https://crm.tradecapitalmarket.com/login
const signupUrl = https://crm.tradecapitalmarket.com/register
```
